Compute bigram frequency of a sentence with a file as input

```
"the quick brown fox and the quick blue hare" => {
    "the quick": 2,
    "quick brown": 1,
    "brown fox": 1,
    "fox and": 1,
    "and the": 1,
    "quick blue": 1,
    "blue hare": 1,
}
```

Run `./run.sh`to build and run app in docker
